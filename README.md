# YouTube-DL-Python
It's a small project which is about youtube video downloading and converting into mp3 built by Python3.

## Environment
python 3.7.7

## import
ssl
pytube 3
moviepy

## Folders

### /Download-mp4
影片載下來以後，會自動放到這裡

### /Download-mp3
影片載下來以後，會自動轉成mp3格式並存放到這裡

### /Download-PlayList
如果是播放清單，整個清單會存在這裡
